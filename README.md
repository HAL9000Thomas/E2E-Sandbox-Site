# E2E-Sandbox-Site
A simple web app for practising test automation with tools like Cypress or Playwright.

## Setup
Simply put all pages within a folder. Navigate inside the folder in your console and run:

```bash
npx serve . -p 3000
```

The pages can then be accessed in your browser at http://localhost:3000/

> Requires [Node.js](https://nodejs.org) to be installed.
